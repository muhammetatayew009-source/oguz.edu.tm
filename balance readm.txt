Внесенные изменения
Интеграция с базой данных и API (main.py):

Добавлена ​​интеграция с SQLite для постоянного хранения и отслеживания балансов пользователей.
Добавлена ​​новая конечная точка /api/order для решения проблемы отсутствия API при взаимодействии пользователей с сайтом.
Создан /api/balance/{username} для динамического получения балансов пользователей.
Создан /api/buy_with_balance, который списывает необходимую сумму непосредственно с баланса пользователя и немедленно предоставляет ключ подписки на VPN без необходимости повторной оплаты в USDT.
Команды администратора Telegram-бота (main.py):

Обновлен ваш веб-хук Telegram! Теперь вы можете ввести команду /addbalance @username amount непосредственно в свой бот.

Пример: /addbalance @niker 100.
Бот ответит, подтверждая пополнение и новый баланс пользователя.

Отображение баланса и логика интерфейса (index.html):

Добавлена ​​небольшая кнопка «Профиль/Вход» в панель навигации. При нажатии на неё запрашивается имя пользователя в Telegram (@username).
Имя пользователя надёжно сохраняется в локальном браузере.
Текущий баланс пользователя в TMT красиво отображается в панели навигации.
При нажатии на кнопку «Satyn al» (Купить):
Если у пользователя достаточно средств на балансе, ссылка на VPN мгновенно генерируется на экране через /api/buy_with_balance.
Если баланса недостаточно, пользователь перенаправляется на pay.html для инициирования запроса на пополнение.
Обновления процесса пополнения (pay.html):

Обновлена ​​формулировка, чтобы отразить пополнение баланса для выбранного тарифного плана.
Когда пользователь отправляет уведомление о пополнении в Telegram, оно теперь автоматически включает его @username, чтобы вы точно знали, кому зачислить средства с помощью команды /addbalance.

Как проверить
Перезапустите сервер FastAPI: uvicorn main:app --reload
Откройте index.html в браузере.
Нажмите «Войти» (Içeri gir) в правом верхнем углу и введите свое имя пользователя Telegram.
Перейдите к своему Telegram-боту и отправьте /addbalance @your_username 200.
Обновите страницу — вы увидите, что ваш баланс обновится до 200 TMT.
Попробуйте приобрести тариф Premium (150 TMT). Это должно сработать, стоимость будет списана (новый баланс 50 TMT), и ваш ключ VPN отобразится немедленно!
